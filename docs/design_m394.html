Шаблоны кнопок

Состояния

Рецепт: 1px обводка + два box-shadow (наружное свечение + inset). Без blur-фильтров — дёшево для слабых телефонов. Тач-таргеты ≥44px (R36). Фиолетовый — только ИИ (R19).

Дерево иснада · связи прорисовываются сами (как в референс-видео)

Связи — SVG path в два слоя (тонкая линия + широкая полупрозрачная «аура»), самопрорисовка stroke-dashoffset 2с. Узлы: Пророк ﷺ — золото, مدار — фиолетовое свечение, شك الراوي — жёлтый. drop-shadow только на 2-3 узлах, не на списках.
