ⓘ Как искать
• обычно — пишешь слова, ищет где есть ВСЕ слова (огласовки игнорируются).
• "точная фраза" — в кавычках: слова строго подряд.
• | — ИЛИ: الصبر | الصلاة найдёт где есть любое.
• -слово — исключить: الصلاة -الجمعة.
• номер — напиши число (напр. 87): покажет хадис №87 из выбранных сборников (или из всех).
• َ Огласовки точно — искать ровно с огласовками как ввёл.
• جذ Однокоренные — ищет однокоренные (приблизительно).
• 🧠 По смыслу — опиши смысл (рус/eng), подберёт арабские фразы.
• Искать в: ничего не выбрано = искать ВЕЗДЕ; выбрал сборники = только в них.
Примеры — нажми, чтобы попробовать:
الصبر «إنما الأعمال» الجار | اليتيم № 87 аят 2:255
