🆕 Muslimoon v399 — конструктор иснада влит в режим «Цепь» (единообразие)

⛓ По замечанию владельца: конструктор не отдельное окно, а часть привычного режима «Цепь». В боксе цепи теперь три порядка поиска: «вместе в иснаде» · «рядом» · «🧱 строго по порядку (конструкция)» — последний ищет цепи, где звенья стоят ПОДРЯД ровно в порядке ввода (1-й передаёт от 2-го…), мгновенно по реестру. Кнопка меню «Поиск по иснаду» просто включает этот режим. Одна кнопка «Найти цепь» — для всех трёх порядков.
