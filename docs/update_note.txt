🆕 Muslimoon v406 — ветки одного хадиса по клику

🌿 В «Таблице всех хадисов» номера стали кликабельными: тап по «версия V…» показывает все точные копии этой версии, а тап по «хадис U… 🌿» — все ВЕТКИ этого уникального хадиса (один хадис, переданный разными иснадами в разных сборниках объединяется воедино). Например один хадис встречается 907 ветками по всем первоисточникам — теперь это видно одним нажатием.
