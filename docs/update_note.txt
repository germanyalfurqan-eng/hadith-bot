🆕 Обновление v300

🎯 Выбор источников для поиска — теперь как надо (по совету и нашей доработке):
• Список разделов, у каждого ➕ → ✓ (весь раздел) / ◐ (выбраны отдельные книги) / ➕ (пусто).
• Блок «🎯 Выбрано» — «📋 список»: пробежаться по выбранным книгам, лишнюю убрать кнопкой «−», сортировать по разделам или авторам.
• Счётчик «Источников: N книг · M разделов».
• Клик по названию раздела — листать его книги, у каждой свой ➕.
Дальше → «🔍 Искать» (ищет только в выбранных). Ничего не выбрал — везде.
