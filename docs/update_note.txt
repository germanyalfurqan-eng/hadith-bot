🆕 Muslimoon v401 — формы передачи разделены точно (наперёд, без двойной работы)

🧬 По твоему вопросу «разделять ли حدثنا/حدثني/أخبرنا/أنبأنا разными кодами»: сделал правильно — теперь у каждого звена хранится ТОЧНАЯ форма (حدّثنا · حدّثني · أخبرنا · أخبرني · أنبأنا · سمعت · عن · أنّ · قال), а в приложении она показана между именами с пояснением способа передачи (سماع / قراءة / إجازة / عنعنة).

🔑 Код цепи группирует формы по юридическому КЛАССУ تحمل: سماع (حدثنا/حدثني/سمعت) · قراءة (أخبرنا) · إجازة (أنبأنا — слабее!) · عنعنة (عن — тадлис). Захотим различать тоньше — поменяем одно правило без перепрохождения всех книг. Это и есть «не делать двойную работу».
