🆕 Muslimoon v411 — порядок в карточке хадиса (тахридж = один хадис)

🧹 Навёл порядок в блоке «Где ещё передаётся»:
• Вверху — научный تخريج учёных (sunnah.one): «Тот же хадис — версии одного хадиса» с номерами по сборникам (это и есть один уникальный хадис).
• Ниже — наш поиск ПО ТЕКСТУ, чётко разделён: «✓ тот же хадис» отдельно, а «≈ похожие по тексту» свёрнуты с предупреждением «могут быть ДРУГИЕ случаи, не один хадис» — чтобы похожие куски не выдавались за тот же хадис.

Это по выработанной формуле уникального хадиса (один случай = по тахриджу учёных, а не по совпадению слов).
