🆕 Muslimoon v414 — 🌿 Тахридж теперь ЕДИНЫЙ ИСТОЧНИК ИСТИНЫ для версий хадиса

🔗 «Где ещё передаётся» в карточке хадиса полностью переведён на НАШ тахридж: версии берутся только из выверенной базы по 64 первоисточникам (раньше — текстовый поиск, мог подмешивать похожие, но другие случаи). Карточка сама опознаёт хадис по началу матна (первые 10 значимых слов после упоминания Пророка ﷺ) и мгновенно показывает все его версии по сборникам — клик по № открывает читалку. Если хадиса нет в тахридже (асар, слова сподвижника, тафсир) — честно говорим «не найдено», без подмеса.

📒 «Таблица всех хадисов» — у каждого хадиса теперь бейдж «🌿 хадис №… · N версий» из того же тахриджа: видно, к какому уникальному хадису (одному случаю Пророка ﷺ) относится запись, тап раскрывает все версии по сборникам прямо в таблице.

Один и тот же тахридж теперь везде: раздел «Тахридж», карточка хадиса и таблица читают одну базу — 240 886 уникальных хадисов.
