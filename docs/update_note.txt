v426 — тахридж больше не показывает сам хадис как «другую версию»

Доводка v425: для карточек Мактабы (mk) refOf даёт поле bid (не source), и фильтр самоисключения пропускался → Бухари 666 показывал «صحيح البخاري ط السلطانية №666» (сам себя). Теперь учитываются обе формы ref (source и bid) — сам хадис исключён из списка версий во всех типах карточек.
