🆕 Обновление v210 — подсветка ≠ чтение по словам + избранное по умолчанию

📖 «Чтение по словам» теперь отдельный режим: КАЖДОЕ слово произносится отдельно (специальная запись) —
   для заучивания. А «✨ Подсветка слов при чтении» — отдельная галочка (подсветка при обычном чтении любым чтецом).
⭐ В «Избранное» по умолчанию: Ислам Камиль (наша, скоро), аль-Мухаймин, Мукбиль, аль-Аʿзами — можно убрать.
🧹 Убрал тумблер «без пауз» (паузы между аятами и так убираются всегда).
