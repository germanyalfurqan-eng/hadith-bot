v427 — передатчики хадиса теперь из ТАБЛИЦЫ кодификации (а не угадываются)

Карточка/читалка хадиса: подсветка и клик по передатчику берут ТОЧНЫЕ id из таблицы кодификации (шарды codif/), а не угадывают по буквам. Пример: Бухари 789 — «أبو بكر بن عبد الرحمن» теперь открывает id 224 (реальный таби'и), а не ошибочного «أبو بكر العنسي» (مجهول). Подсвечиваются только реальные передатчики цепи этого хадиса.
