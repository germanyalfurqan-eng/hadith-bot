🆕 Обновление v272 — Переключатель «Строгий поиск» в Настройках

🎯 В ⚙️ Настройках появился тумблер «Строгий поиск: только где есть ВСЕ слова». По умолчанию ВКЛючён (точные совпадения — показываются только хадисы, где присутствует каждое слово запроса). Выключишь — поиск покажет и близкие/частичные результаты.
