v216 — Пачка правок: ① сортировка корней «по формам» теперь работает (группирует аяты по форме слова с заголовками); ② кнопка «закрыть» во всех окнах крупнее и попадаемее; ③ у кнопки «структурировать» появилась подсказка ЧТО она сделает; ④ результаты поиска по Мактабе можно ФИЛЬТРОВАТЬ по разделам (тафсир / كتب السنة / …, мультивыбор); ⑤ размер шрифта Корана теперь отдельный — не меняет шрифт в других местах; ⑥ при переходе аята звук берётся из предзагрузки (без спиннера-загрузки).
