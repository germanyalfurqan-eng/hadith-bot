🆕 Muslimoon v413 — 🌿 ПОЛНЫЙ ТАХРИДЖ ОТ А ДО Я (уникальные хадисы по случаю)

🌿 Новый раздел в меню «☰ Ещё» → «Тахридж — уникальные хадисы». Прошёл по ВСЕМ 64 первоисточникам (40 сборников + Табари) и сам выстроил тахридж по формуле уникального хадиса: один уникальный хадис = ОДИН случай с Пророком ﷺ, переданный разными путями и формулировками (точь-в-точь / сокращённо / пересказ).

Итог: 269 877 записей → 240 886 уникальных хадисов, из них 15 064 имеют тахридж (несколько версий). Открой — листай уникальные хадисы по числу версий, тап показывает ВСЕ источники каждого (по сборникам, клик по № — читалка). Например «من كذب علي متعمدا» — 63 версии по всем сборникам.

Проверено: марфу'-хадисы объединяются верно (من قام رمضان, التحيات, صلاة في مسجدي…), асары и тафсир не сливаются ложно по иснаду.
