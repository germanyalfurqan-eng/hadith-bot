🆕 Обновление v188 — интерактивное чтение Корана с подсветкой по словам (β)

✨ В читалке Корана кнопка «Читать по словам»: слово подсвечивается по мере чтения (тайминги quran.com).
   Доступно для чтецов Афаси, Хусари, Абдуль-Басит, Миншави, Шурайм. Для остальных (включая Худайфи по
   умолчанию) включается Афаси для подсветки — обычное чтение по аятам тоже работает как раньше.
