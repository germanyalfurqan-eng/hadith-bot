# 📖 Руководство по боту Muslimoon — простыми словами

Пиши команды боту обычным текстом. Регистр не важен.

---

## 1. Аль-Мухаймин (الموحد المهيمن) — наша выверенная книга

| Что написать | Что получишь |
|---|---|
| `мухэймин 13` (или `муршид 13`) | Хадис №13: книга, глава, все его версии (риваяты), каждый — отдельным сообщением |
| `книги` | Список всех 44 книг с диапазонами номеров |
| `книга 5` или `книга الصيام` | Список глав книги с номерами хадисов |

**Как читать карточку хадиса:**
- 📕 — книга (كتاب), 📂 — глава (باب)
- `▫️ Риваят 1/3 ✅ [أحمد ٠37]` — это версия 1 из 3:
  - **✅** = текст сверен с первоисточником, **⏳** = пока не сверен
  - **`[أحمد ٠37]`** = *метка автора* — как Муршид сослался в книге. ⚠️ Цифры в метке иногда искажены OCR (могут быть неточными).
  - **`🔖 первоисточник: ahmad 960`** = *выверенный стандартный номер* (мы нашли по тексту). **Это надёжный номер.**
- Если видишь два разных числа (напр. метка `٠37` и `ahmad 960`) — это ОДИН хадис: первое число — авторская ссылка с кривой OCR-цифрой, второе — правильный номер первоисточника.

## 2. Достоверность хадиса (sunnah.one) — 🔥 проверка хукма

| Что написать | Что получишь |
|---|---|
| `сунна من غشنا فليس منا` | Хадис + оценка достоверности + тахридж |
| `достоверность احفظ الله` | то же |
| `хукм انما الاعمال` | то же |

Показывает топ-5 совпадений, для каждого:
- ✅ / ⚠️ / ℹ️ **الحكم** — оценку (`صحيح` / `ضعيف` / `إسناده صحيح على شرط مسلم` и т.д.) **от разных мухаддисов**
- 📋 **التخريج** — где хадис выведен (أخرجه الترمذي ٢٥١٦…)

👉 Это лучший способ проверить любой хадис. Ищи по куску текста (матну).

## 3. Хадисы из 8 сборников (по номеру)

`бухари 33` · `муслим 16` · `абу дауд 1` · `тирмизи 1` · `ибн маджа 1` · `насаи 1` · `муватта 1` · `ахмад 1`

Даёт: 🔤 арабский + 🌍 русский перевод + 📊 оценку (если есть) + отметку, если хадис есть в аль-Мухаймине.

⚠️ **Муслим:** у источника нумерация местами пустая/нестандартная (مقدمة и пр.) — некоторые номера не находятся. Надёжнее искать Муслима по тексту через `сунна <текст>`.

## 4. Первоисточники → где в аль-Мухаймине

Для сборников без своего раздела (Таялиси, Хумайди, Ибн Аби Шейба, Дарими, Абу Я'ла и др.):

`тиялиси 323` · `хумайди 28` · `ибн аби шейба 100`

→ покажет сам риваят + в каких номерах аль-Мухаймина он встречается.

## Частые вопросы

**«Почему в метке одно число, а в первоисточнике другое?»**
Метка `[أحمد ٠37]` — ссылка автора (цифры искажены OCR). `🔖 ahmad 960` — правильный стандартный номер. Это один хадис.

**«Бухари выдал без арабского»**
Бывает таймаут загрузки текста с CDN. Теперь есть повтор и резервный источник — должно стать стабильнее. Если повторится — просто повтори запрос.

**«Муслим выдаёт не тот хадис / пусто»**
Нумерация источника Муслима нестандартна. Используй `сунна <текст>` — найдёт по тексту с достоверностью.
