🆕 Обновление v101 — разбор слова и корня как corpus.quran.com, но ПО-РУССКИ 🌿

📝 Тап по слову аята теперь показывает ГРАММАТИКУ на русском: часть речи, порода
   глагола (I–X), время, лицо/род/число, падеж, лемму. Пример: يؤمنون →
   «глагол, порода IV, наст./буд. время, 3-е л. муж. мн.ч.; корень أمن».

🌿 «Все места корня» → теперь с разбивкой ПО ФОРМАМ (как corpus.quran.com):
   сколько раз встречается корень, в каких формах (رحيم — прил. 116, رحمة — имя 114,
   رحم — глагол I 29 …), клик по форме = фильтр. Переключатель сортировки:
   «↕ по порядку сур» / «🔤 по формам».

🔢 Ввод номера аята вроде «235» больше не «убегает»: показывает варианты
   (2:35 — аль-Бакара / 23:5 — аль-Муминун), нажми нужный.
