🆕 Обновление v107 — поиск-помощник перестал «лить бред» + ИИ предлагает фразу из хадиса

🔤 Раньше на русскую фразу (напр. «дела по намерениям») подсказки выдавали мусор
   (نعمركم·ضل·العدل). Теперь по-арабски подсказывается ТОЛЬКО когда реально похоже на
   транскрипцию (бида→بدعة, сабр→الصبر, халифату расуль→خليفة الرسول). Русские фразы со
   словами «по/в/что…» — не трогаем (их ведёт «🧠 По смыслу»).

🧠 «По смыслу (ИИ)» теперь для описания хадиса подбирает ХАРАКТЕРНУЮ ФРАЗУ из самого
   текста: «дела по намерениям» → إنما الأعمال بالنيات, «иджтихад судьи» → إذا حكم الحاكم فاجتهد.
