🆕 Обновление v137 — Коран ищет по корню, когда точного слова нет

🌿 Раньше поиск «البدعة» в Коране давал «ничего не найдено» (точного слова в Коране нет). Теперь
   приложение показывает аяты по КОРНЮ слова: «بدعة» → корень بدع → аяты с «بديع / ابتدعوها / بدعا»,
   с пометкой, что показано по корню.
