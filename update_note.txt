🆕 Обновление v208 — точное чтение по словам из quran.com (для их чтецов)

🎯 Если выбран чтец с пометкой «✨ подсветка» (Афаси, Хусари, Абдуль-Басит, Миншави, Шурайм) — обычная
   кнопка ▶ читает аят ТОЧНО по словам по разметке quran.com (их запись + готовые таймкоды), без пауз
   между аятами. Для Худайфи (его нет на quran.com) — подстройка под голос (как раньше).
   Хочешь точное по словам — выбери чтеца с «✨» в «⚙️ Настройки чтения».
